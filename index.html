DESARROLLO DE DISTROS LINUX

Una distribucion Linux es una distribucion de software basada en el nucleo Linux, y a menudo, un Sistema de gestion de paquetes que incluye determinados paquetes de software, para satisfacer las necesidades de un grupo especifico de usuarios.

AUDITORIA INFORMATICA

Es un proceso en el cual se hace un analisis a toda la infraestructura de los recursos IT de una empresa. Esto permitira conocer el estado de la seguridad actual de esta y asi eliminar las vulnerabilidades y asi elevar el nivel de proteccion de los recursos informaticos de la empresa.

PROTECCION DE DATOS

Es el proceso de salvaguardar informacion importante contra corrupcion, filtraciones, perdida o compromiso de los datos de una persona o empresa.

ANALISIS FORENSE

Es el proceso por el cual se recopilar, examinar y analizar la evidencia digital con el fin de determinar como ha ocurrido un incidente de seguridad o un delito informatico.

ANALISIS DE VULNERABILIDADES

Es el proceso en el que se identifican y clasifican las debilidades de un sistema informatico con el objetivo de dar una vision de las amenazas en las que puede incurrir una empresa y como evitarlas a tiempo.

PENTESTING

ES el proceso por el cual se hace un ataque malicioso simulado contra los sistemas informaticos que se usa para encontrar y verificar posibles vulnerabilidades.

CRIPTOGRAFIA

Es el proceso por el cual se busca proteger informacion mediante el uso de algoritmos codificados, hashes y firmas.
